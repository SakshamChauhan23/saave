# Product Requirements Document

## Problem Statement
Knowledge workers save valuable content across dozens of platforms.
Every platform has its own bookmark system, leading to fragmented knowledge,
duplicate saves, and almost zero revisits.

## Solution
A universal capture platform consisting of:
- Responsive PWA (primary experience)
- Chrome Extension
- iOS Share Extension
- Android Share Target
- Unified backend and inbox

## Workflow
1. Discover content
2. Share or capture
3. Metadata extraction
4. Store as Knowledge Asset
5. View in unified inbox

## Feature List
1. Universal Inbox
2. Universal Capture
3. iOS Share Extension
4. Android Share Target
5. Chrome Extension
6. Search
7. Authentication
8. AI Metadata Extraction
